To Hide Featured Image on individual posts.

== Description ==

There are instances when one wants to show featured image on one post and hide it on another post.
This plugin will allow you to show or hide featured image from single post page. It supports custom post types also.


== Installation ==

1. Upload `hide-featured-image.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How to use it =

Open the post in edit mode. Select the checkbox to hide featured image and update.
